# Contributing

## Branch naming

Prefix every branch with one of the following, followed by the related
issue number and a short description:

| Prefix | Use for |
|---|---|
| `feature/` | New functionality |
| `fix/` | Bug fixes |
| `chore/` | Maintenance, dependency updates, etc. |
| `docs/` | Documentation only |
| `test/` | Adding or fixing tests |
| `ci/` | CI/workflow changes |
| `refactor/` | Code restructuring with no behavior change |
| `perf/` | Performance improvements |

Format: `<prefix>/<issue-number>-<short-description>`

Example: `feature/12-template-notes`, `fix/8-wikilink-escaping`.
